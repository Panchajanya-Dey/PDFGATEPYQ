

## For First-Time GitHub Users

### What is GitHub?
GitHub is a platform where developers share code. If you're new to GitHub, don't worry! Here's how to get this project on your computer:

### Step 1: Download the Code
You don't need to use Git commands if you're not comfortable with them. There are two easy ways:

#### Option A: Download as ZIP (Easiest for beginners)
1. Look for a green button that says **"Code"** at the top of this page
2. Click it and select **"Download ZIP"** from the dropdown menu
3. Once downloaded, extract (unzip) the folder to a location you'll remember (like your Desktop)
4. You're done! You now have the project files on your computer

#### Option B: Use GitHub Desktop (If you want to learn Git)
1. Download [GitHub Desktop](https://desktop.github.com/) for free
2. Install and open GitHub Desktop
3. Click **"File"** → **"Clone Repository"**
4. Paste this repository URL: `[https://github.com/Panchajanya-Dey/PDFGATEPYQ.git](https://github.com/Panchajanya-Dey/PDFGATEPYQ.git)`
5. Choose where to save it and click **"Clone"**

### Step 2: Install Python (If you don't have it)
This project requires Python to run. Here's how to check if you have it:

#### Check if Python is installed:
- **Windows**: Open Command Prompt and type `python --version`
- **Mac/Linux**: Open Terminal and type `python3 --version`

If you see something like `Python 3.8.5` or higher, you're good to go!

#### If Python is NOT installed:
- **Windows**: 
  1. Go to [python.org/downloads](https://www.python.org/downloads/)
  2. Click the yellow **"Download Python"** button
  3. Run the installer and **IMPORTANT**: Check the box that says **"Add Python to PATH"** at the bottom
  4. Click "Install Now"

- **Mac**: 
  1. Open Terminal
  2. Install Homebrew first: `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
  3. Then run: `brew install python`

- **Linux (Ubuntu/Debian)**:
  ```bash
  sudo apt update
  sudo apt install python3 python3-pip python3-venv
  ```
### Step 3: Launching Locally
  1. Go to the folder where you have downloaded the code. 
  2. Follow instructions from README.md for launching the website locally.
